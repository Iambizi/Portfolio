Application built with the Poke API data. The application is a dashboard of the original 150 pokemons from the series.
Upon clicking on a pokemon you can view more information and stats around it.
No authentication was required to access this API.

Recipe application built using the Food2Fork API. You can search for different recipes based on food names.

Photo application built using the Unsplash API. Each image showcases the unsplash user's name who uploaded it and number of likes the picture received.
You may also click a download link that will take you to the unsplash site for download.
