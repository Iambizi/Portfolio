VIDLY application

View Project View Code

Movie rental application built with React.js.
The application is set up with PAGINATION, FILTERING and SORTING features for a list of movies coming from a database and ROUTING for the different pages of the app.
You are also able to add or edit new movies via a FORM.
Authorization and AUTHENTICATION is also present on the application to allow for users to first register or login to edit or add movies.
The application is CONNECTED TO A BACK-END hosted on a MongoDB database hosted on mLab.
